# Changelog

All notable changes to StructuraMiner will be documented in this file.

## [1.0.0] - 2026-06-08

### Added
- Initial release of StructuraMiner
- Exhaustive structural feature extraction from PDB files
- 7 output modules
- Support for both full RCSB and stripped ATOM-only PDB files
- Multi-core parallel processing
- Auto-dependency installation
- ML-ready CSV output format
- Comprehensive feature set including SASA, dihedrals, secondary structure, interactions, and more

### Features
- Zero configuration required
- Automatic PDB format detection
- Detailed extraction summary report
- Command-line interface with multiple options
